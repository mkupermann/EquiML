Installation
============

To install EquiML, follow these steps:

1. Clone the repository:
   .. code-block:: bash

      git clone https://github.com/mkupermann/EquiML.git
      cd EquiML

2. Install dependencies:
   .. code-block:: bash

      pip install -r requirements.txt

3. Install EquiML:
   .. code-block:: bash

      pip install .

Dependencies include scikit-learn, Fairlearn, SHAP, Matplotlib, Seaborn, and Streamlit.
