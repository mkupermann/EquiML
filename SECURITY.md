# Security policy

## Reporting a vulnerability

Email security reports to **michael@kupermann.com**. Include:
- A description of the vulnerability and the affected version.
- Steps to reproduce.
- The impact you observed.

I will acknowledge within 5 business days and aim to resolve or mitigate
within 90 days. EquiML is a single-maintainer project, so response is
best-effort.

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Disclosure policy

Standard 90-day coordinated disclosure. If a fix is not feasible within
that window, I will request an extension before public disclosure.
